# DairyOps Store Listing Draft

## App Name

DairyOps

## Short Description

Manage dairy herd health, milk production, inventory, and farm cash flow.

## Full Description

DairyOps helps dairy farms manage everyday operations from one clean mobile workspace.

Track animals, record milk production, monitor herd health, manage feed and medicine inventory, and review farm income, expenses, profit, and available cash. The dashboard highlights daily production, monthly financial flow, low-stock items, and management insights so farm owners can make faster decisions.

Core features:

- Herd and animal profile management
- Milk production records
- Health and vaccination visibility
- Sales and expense tracking
- Inventory and low-stock monitoring
- Cash flow and monthly farm summaries
- Secure login for farm accounts

## Data Safety Notes

Expected data types:

- Account information: username, email, farm name
- App activity: farm records created by the user
- Financial information: sales, expenses, withdrawals, loans, inventory costs
- User-generated content: animal notes, inventory notes, transaction descriptions

Use HTTPS for production API traffic. Add a public privacy policy URL before publishing.
